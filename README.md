## Features
- Reading sensors using **ADC** & **I2C**
- Evaluating data with an **Embedded AI**
- Store the data on **SD Card**
- Serve **Live Data** via **Local Website**
- Giving **Audio Feedback** using **DAC**
- Controlling **LED Brightness** via **PWM**
- Having a **Menu Interface on LCD**
- Maintain **USB Comminication** with PC
